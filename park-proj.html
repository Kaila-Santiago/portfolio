Project Overview

Skills Exercised: Copyediting, Rewriting, Proofreading, Grammar Conventions, Sentence Structure

Goal: To revise the event description for "The Need You Know It Is a Letting Light" on the Prospect Park Alliance website to improve clarity, voice, and flow without compromising the intended message.

Background: This project was done for my Professioal Editing and Revising course. To revise the art installation description, I assessed each sentence to identify:

  • Sentence type (simple, compound, complex, compound-complex)
  • Verbs (tense, active/passive, strength)
  • Sentence structure

In my assessment, I found that some sentences were overly complex, which made it difficult to follow main ideas. Out of the six sentences, I copyedited five. This page includes the completed project, a side-by-side comparison of each sentence, and a breakdown of changes made.
